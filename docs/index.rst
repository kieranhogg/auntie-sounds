auntie-sounds documentation
===================================

This is a library for interacting with BBC radio stations via an interface to BBC Sounds. 
It was written in mind to be used for the BBC Sounds provider 
for `Music Assistant`_  but exists as a standalone library.

.. _Music Assistant: https://music-assistant.io/
