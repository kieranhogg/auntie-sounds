auntie-sounds documentation
===================================

This is a library for interacting with BBC radio stations via an interface to BBC Sounds. 
It was written to be used for the BBC Sounds provider for [Music Assistant](https://music-assistant.io/) but exists as a standalone library.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.
